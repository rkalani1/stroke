# Obtain blood draw STAT for:

* + Emergency Stroke Panel (includes PT/INR, a Direct Xa Inhibitor Screen, thrombin time (TT), fibrinogen, CBC and PTT)
  + Type and Screen – STAT
  + If crash craniotomy considered: 2 Units emergency-release RBCs (include both prepare and transfuse orders).

1. **Obtain medical history:**
   * Specific anticoagulant/antiplatelet agent
   * Last dose / timing
   * Duration of therapy
   * Medical comorbidities (e.g. renal or hepatic dysfunction) or drug-drug interactions that may impact antithrombotic agent metabolism
   * Indication for anticoagulant
2. **Warfarin:**

* Give vitamin K 10 mg IV
* INR-Based Reversal:

|  |  |  |
| --- | --- | --- |
| INR Level | Reversal Strategy | COR / LOE |
| ≥2.0 | 4-factor PCC (preferred) | 1 / B |
| 1.6-1.9 | 4-factor PCC recommended | 2b / C |
| 1.3-1.5 | Consider 4-factor PCC case-by-case | 2b / C |

COR = Class (Strength) of Recommendation & LOE = Level (Quality) of Evidence

*Absolute contraindications to 4-factor PCC:*

* Disseminated intravascular coagulation (DIC)
* Heparin induced thrombocytopenia (HIT)

*Relative contraindications (potential benefits usually outweighs potential risks, discuss with stroke phone):*

* Thrombotic event <6 weeks
* Known prothrombotic condition
* Major surgery <6 weeks
* Mechanical circulatory support (consult Cardiology MCS attending)

# *4-Factor PCC/KCentra Dosing:*

# Infuse 2000 units IV immediately

# Repeat PT/INR at 30 minutes and every 6h for 24h after PCC infusion

# If PT/INR>1.5 after infusion, page hematology and consider administering 500 units of PCC or 2-4 units of plasma (FFP)

# If INR>1.5 at 24 hours, repeat vitamin K 10 mg IV dose

# If PCC unavailable or contraindicated, administer Plasma (FFP): Place prepare and transfuse orders for 4 units emergency-release plasma immediately and request an additional 4 units (note: ideal plasma dose is 15 mL/kg and each unit is 250-300 mL, so be aware some patients may need additional units).

# Consider concurrent diuretic (furosemide) if history of CHF

# Repeat STAT emergency stroke panel immediately after completion of infusion

# Repeat INR every 6h for 24 h after plasma

# If INR>1.5, place prepare and transfuse order and administer 4 units plasma again & consult hematology attending

# If INR >1.5 at 24 hours, repeat vitamin K 10 mg IV dose

1. **Dabigatran** (direct thrombin inhibitor)

COR/LOE: 2a / B

* Assessment: Thrombin Time (TT)
  + Normal TT excludes significant dabigatran effect
* Reversal:
  + If ingestion <2 hours à activated charcoal (oral)
  + Idarucizumab (Praxbind) 5g IV (give as 2 doses of 2.5g, ≤15 minutes apart; each dose infused over 5-10 minutes)
  + If Idarucizumab unavailable à 4-factor PCC 2000 units IV
  + In the setting of renal failure, consider emergent dialysis

1. **Rivaroxaban, Apixaban, or Edoxaban** (Factor Xa inhibitors)

COR/LOE: 2b / B

* Assessment: Direct Xa Inhibitor Screen
  + Normal screen excludes significant anticoagulant effect
* Reversal:
  + If ingestion <2 hours à activated charcoal (oral)
  + 4-factor PCC 2000 units IV
  + In the setting of renal failure, consider emergent dialysis

1. **Treatment-dose unfractionated heparin**

COR/LOE: 2a / C

* Assessment: Anti-Xa
* Reversal:
  + Protamine 25 mg IV immediately
  + Assess anti-Xa after infusion
  + If anti-Xa > 0.1 à administer additional 10 mg protamine (max cumulative dose: 55 mg)
  + Additional assessment: if platelets are < 100 THOU THOU/µL, send for Heparin Induced Platelet Antibodies to screen for heparin induced thrombocytopenia (HIT). If positive, consult hematology.

1. **Treatment dose low-molecular-weight heparin**, [enoxaparin, dalteparin, or tinzaparin] ≤ 8 hours

COR/LOE: 2b / C

* Assessment: clinically significant hemorrhage
* Reversal:
  + Last dose <8 hours: Protamine 50 mg IV
  + Last dose 8-24 hours: Protamine 25 mg IV
  + Last dose >24 hours: no reversal indicated

1. **Post-IV thrombolytic (Alteplase / Tenecteplase)**

[See OCCAM Protocol]

1. **Antiplatelet Agents** (aspirin, clopidogrel, prasugrel, ticagrelor, NSAIDS, glycoprotein IIb/IIIa antagonists [abciximab, eptifibatide, or tirofiban]):
   * Discontinue antiplatelet agent (s)
   * Platelet transfusion: NOT routinely recommended (COR/LOE: 3 / B, harm demonstrated in PATCH trial)
2. **Other antithrombotic treatment**, stop the agent and STAT [consult hematology ATTENDING. Such](https://depts.washington.edu/uwstroke/stroke_rx/IV_tPA_incl_excl.pdf) agents would include pentasaccharides such as fondaparinux and direct thrombin inhibitors (DTIs) such as, bivalirudin, or argatroban. There are no specific ‘antidotes’ for these parenteral DTIs or for fondaparinux. Bivalirudin, and argatroban have short-half lives. Repeat DTI level at 60 minutes after stopping and if it is still prolonged consult hematology ATTENDING. Fondaparinux has a very long half-life, influenced by renal function.
3. Notify Stroke attending either directly (206 744 6789) or via the Transfer Center if at UWMC-NW (206-520-7575) about the patient and Neurosurgery if needed. If surgery is considered, call the Transfusion Service and request 2 Units cross-matched RBC (takes approx. 5-50 minutes; at UWMC-NW for no current type & screen available it may take up to 3 hours; for patient with red cell antibodies call lab to determine ETA). If crash craniotomy is considered, request 2 Units of emergency release RBC and place prepare and transfuse orders.
4. If prolonged PTT is the only abnormality identified on the STAT blood(Step 1 of this protocol), immediately consult hematology ATTENDING through the paging operator.
5. Request additional products, as below, if not already requested as part of protocols above, based on the blood results and assuming an average-sized adult.

* If INR > 1.5, see above recommendations (#3) for use of 4-factor PCC or plasma.
* If fibrinogen <125 mg/dL, place prepare and transfuse order for 2 units of cryoprecipitate (1 unit = 5 pre-pooled cryoprecipitates).
* If platelets <50 THOU/µL, place prepare and transfuse order for 2 units of platelets.
* If platelets 50 to 100 THOU/µL, place prepare and transfuse order for 1 unit of platelets.

1. Complete your facility’s IPH order set and document ICH severity score in your note with correct time stamp or using the Stroke tab in EPIC.

## **Laboratory Monitoring**

|  |  |
| --- | --- |
| *Time Point* | *Tests Required* |
| Baseline | Emergency Stroke Panel |
| 15-30 min post-reversal | Repeat STAT Emergency Stroke Panel [includes PT/INR (warfarin), TT (dabigatran), Anti-Xa (factor Xa inhibitors)] |
| 6 hours | PT/INR, CBC |
| 24 hours | PT/INR, CBC, fibrinogen |

## **Imaging**

1. Baseline CT head
2. Repeat CT at 6 hours, sooner if clinical deterioration
3. CT at 24 hours

## **Clinical Monitoring**

* Neuro checks q1h × 24 hours
* Blood pressure per ICH protocol
* Monitor for thrombotic complications

## **Reversal Agents: Efficacy**

|  |  |  |
| --- | --- | --- |
| *Agent* | *Anticoagulant* | *Efficacy* |
| 4-factor PCC + vitamin K | Warfarin (with INR≥2) | INR correction in most |
| Idarucizumab | Dabigatran | >90% TT normalization |
| 4-factor PCC | Factor Xa inhibitors | Similar to andexanet |
| Platelet transfusion | Antiplatelet | Harmful |

Andexanet for Factor-Xa inhibitors à 82% hemostatic effectiveness (COR/LOE: 2a / B) but NOT available at UW Medicine

DDAVP: Efficacy uncertain for management of antiplatelet-associated hemorrhage (COR/LOE: 2b / C)

## **Anticoagulant Pharmacokinetics**

|  |  |  |  |
| --- | --- | --- | --- |
| *Drug* | *Half-life* | *Renal Clearance* | *Dialyzable* |
| Warfarin | 40 hours | No | No |
| Dabigatran | 12-17 hours\* | 80% | Yes (65%) |
| Rivaroxaban | 5-9 hours\* | 33% | No |
| Apixaban | 12 hours\* | 27% | No |
| Edoxaban | 10-14 hours\* | 50% | No |

\*Prolonged in renal impairment
